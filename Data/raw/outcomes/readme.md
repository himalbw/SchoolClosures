The Educational Recovery Project (2020-24): https://edopportunity.org/recovery/data/downloads/

File Used:  seda_admindist_long_gys_2024.3

Unit: Administrative school district

Form: Long (district × year × grade × subject)

Metric: GYS (Grade-within-Year standardized scale)
